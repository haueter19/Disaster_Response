#### Purpose

### Prereqs
<ul>
  <li>Python 3.7</li>
  <li>Pandas</li>
  <li>Plotly</li>
  <li>Flask</li>
  <li>NLTK</li>
  <li>SQL Alchemy</li>
  <li>Scikit Learn</li>

### How To
<ul>
  <li>Install the requisite packages</li>
  <li>From command line, navigate to app folder then python run.py</li>
  <li>Open a browser to http://localhost:3001 or http://0.0.0.0:3001</li>
</ul>
